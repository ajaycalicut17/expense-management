# Expense Management

## Requirement
#### PHP Version: 8.5.5
#### Composer Version: 2.9.7
#### Node Version: 24.14.1

## Installation

```bash
git clone https://github.com/ajaycalicut17/expense-management.git
```

```bash
cd expense-management
```

```bash
composer install
```

```bash
npm install
```

```bash
cp .env.example .env
```

```bash
php artisan key:generate
```

```bash
DB_DATABASE=laravel
DB_USERNAME=sail
DB_PASSWORD=password
```

```bash
php artisan migrate --seed
```

```bash
npm run dev
```

```bash
php artisan serve
```

```bash
Login URL: http://localhost
```
